# LRT Epika Kodi Plugin

Kodi plugin for streaming content from LRT Epika (https://epika.lrt.lt), the Lithuanian National Television streaming service.

## Requirements

- Kodi 19+ (Matrix or newer)
- Python 3.x
- Internet connection
- For DRM content: inputstream.adaptive addon
